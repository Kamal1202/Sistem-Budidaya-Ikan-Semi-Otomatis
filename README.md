# Smart Fish Farming System using Fuzzy Logic

## Overview

This project implements a Fuzzy Logic-based decision support system for fish farming. The system helps users monitor water quality and generate recommendations for fish cultivation activities.

## Features

* Fish Feed Quantity Recommendation
* Pond Water Replacement Recommendation
* Harvest Time Recommendation

## Input Parameters

* Water Temperature
* pH Level
* Water Turbidity
* Dissolved Oxygen (DO)
* Cultivation Duration

## Technologies Used

* Python
* Scikit-Fuzzy
* Pandas
* NumPy
* Google Colab
